Authorization functionality for Microsoft Orleans

The set of security packages for Microsoft Orleans 3.* provide the ability to use the same authorization functionality which is used in ASP.NET Core It allows to use Azure Active Directory or IdentityServer4 v4.0 with MS Orleans grains and any ASP.NET Core 3.* or 5.* application.

Dependencies

There are two packages: Orleans.Security.Cluster and Orleans.Security.ClusterClient. The first is for a silo host project and the seccond for an Orleans cluster client.
